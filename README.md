Employee Performance Dashboard (Excel)

Overview
This project is an interactive Excel dashboard built using Pivot Tables and VBA to analyze employee performance metrics such as efficiency, working hours, and break time.

Features

* Dynamic Pivot Tables and Charts
* KPI tracking (Efficiency, Work Hours, Break Time)
* Interactive dashboard with VBA controls
* Chart show/hide functionality
* Multi-sheet structured analysis

Tools Used

* Microsoft Excel
* Pivot Tables and Charts
* VBA (Visual Basic for Applications)

Use Case
This dashboard helps analyze employee productivity and supports data-driven decision-making.
